Resources

List of resources used in this project.

  • NodeJS - core framework that makes it all work.
  • jspm - jspm package manager.
  • systemjs - systemjs module loader.
  • Gulp - workflow enhancement tool.
  • AngularJS - most popular client side JavaScript framework.
  • angular-route - default angular router.
  • ui-router - router from angular-ui project.
  • angular-new-router - new angular router for 2.x and 1.4+.
  • basap - github repository for this project.
  • jspm-angular-seed - github repository learning jspm in conjunction with angular.
